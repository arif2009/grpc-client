Sample codes for developing a REST back-end service.

# Required tools to run
- Go >= 1.16
- [Docker](https://www.docker.com/products/docker-desktop)

# How to run server
- via docker compose
    ```
    docker compose up -d
    ```
- directly
    ```
    go run cmd/sample/main.go
    ```
